# Media Backup Tool

![Python](https://img.shields.io/badge/Python-3.8%2B-blue) ![Tkinter](https://img.shields.io/badge/Tkinter-GUI-orange) ![Backup](https://img.shields.io/badge/Backup-Automation-green)

## 📌 Overview
**Media Backup Tool** is a Python-based GUI application designed to help users efficiently back up their media files (images & videos). It organizes files based on their creation or modification date, making it easy to manage media backups.

## ✨ Features
- **User-Friendly GUI** – Built with Tkinter for an interactive experience.
- **Backup with Organization** – Automatically categorizes files by year and month.
- **Progress Tracking** – Displays backup progress via a progress bar.
- **File Integrity & Logging** – Prevents overwrites and logs errors for troubleshooting.
- **Supports Various Formats** – Handles popular image and video file types.

## 🛠️ Installation
### 1️⃣ Clone the Repository:
```sh
git clone https://github.com/your-username/media-backup-tool.git
cd media-backup-tool
```
### 2️⃣ Install Dependencies:
```sh
pip install -r requirements.txt
```
### 3️⃣ Run the Application:
```sh
python main.py
```

## 🔧 Configuration
The tool loads supported media file types from `backup_config.json`. You can customize it by adding/removing file extensions.
